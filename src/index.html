Denoising Autoencoder

Artifical Neural Network

The Denoising Autoencoder is a artificial neural network, which can receive a corrupted or noised input and trys to reconstruct the original, uncorrupted data. This decorruption, or denoising, will be learned by the neural network itself and doesn't require human interaction. That kind of Autoencoder can be used to reconstruct for example damaged pictures or videos but can be easily transformed to other types of Autoencoder to automatically encode and decode data (hence the name) for more efficient image or video storage.

  • Date: January 2017
  • Category: A.I.
  • Programming Language: Python
  • Framework: Tensorflow

Clear Up Robot

Robot Behaviour that clears up objects

This behaviour for an AMR (automated mobile robot) will operate in a specific area (known via a map) and perform clear up tasks for specific objects. You simply hold a coloured object in front of the robot's camera and it starts moving to the clear up area. It will then search for colour matching objects and pick them up with its 2-DOF gripper. It then moves towards a deploy area where it stores the obtained objects and then attempts to search for other objects.
This project was made with a classmate at my university's robotics lab.

  • Date: December 2018 - January 2019
  • Category: Robotics
  • Programming Language: C++
  • Framework: Aria, ACTS
  • Robot: Pioneer 2-DX

Big Data meets Container

Containerized Hadoop Cluster

Big Data technologies like Hadoop or Cassandra are designed to run on multiple servers as a cluster. These cluster nodes can be seperately containerized with Docker to provide a platform independet operation. These containers can be deployed and orchestrated throughout a whole set of servers in a Kubernetes cluster. Servers can be dynamically added and removed without the risk of data loss. That makes the handling of huge server clusters much more convenient and modular.
This project was made collaboratively with a classmate at my university as a team project.

  • Date: April - July 2018
  • Category: Big Data, Containerization
  • Programming Language: Bash, Dockerfile, YAML
  • Framework: Docker, Kubernetes

SFML Libraries

Libraries to enhance the usage of SFML

SFMLApp

A library which offers fundamental scene/view support as a kind of node system where drawable nodes can contain other drawable nodes and so on. The goal is to ease the use of architechtural patterns for user interfaces like MVP or MVC.

SFMLMath

SFML offers own vector classes to represent positions or other mathematical units. But only a handful of operations are natively implemented for them. This templated library deals with this problem and adds all commonly used vector operations directly in the SFML-namespace to make the use as SFML-like as possible.

  • Date: February 2019
  • Category: Computer Graphics
  • Programming Language: C++
  • Framework: SFML
